# Летнее бинго — контекст для Claude Code

## Что это

Минималистичный интерактивный статический сайт — дополнение к онлайн-курсу по фотографии (~20 участниц, 3 недели). Участницы «собирают» летние наблюдения, отмечая выполненные задания. Практика намеренно **без камеры** — соединяться с моментами лета, не беря телефон.

## Структура проекта

```
/
├── docs/
│   ├── letnee-bingo-master-tz.md   # Главное ТЗ (источник истины)
│   └── letnee-bingo-code-tz.md    # ТЗ для разработки (этот файл — главный бриф)
└── index.html                      # Единственный артефакт сборки
```

## Техстек

- **Один файл `index.html`**: HTML + CSS + JS инлайн, vanilla JS, без фреймворков, без сборки
- **Хостинг**: GitHub Pages (все пути относительные)
- **Сохранение**: `localStorage` с `try/catch`, ключ `letnee-bingo:v1`
- **Без бэкенда, без аккаунтов, без внешних зависимостей** (кроме шрифтов и опционально `html2canvas` через CDN как фоллбек для PNG-экспорта)

## Ключевые решения (не менять без согласования)

| Тема | Решение |
|------|---------|
| Сетка | 4×5 = 20 ячеек |
| Адаптив | Десктоп — приоритет; мобайл обязателен (2 колонки на узком) |
| Тач-таргеты | ≥ 44px |
| Финал | Фраза «Лето замечено» — тихая открытка, без модального давления |
| Отмена | Долгий тап или маленький контрол на марке — отлепить и вернуть фразу |
| Экспорт | PNG 1080×1920 (сторис 9:16) через offscreen canvas, fallback html2canvas |
| Язык | Только русский |
| Счётчик | Маленький `N / 20` в углу, без давления |

## Данные (контент)

Все фразы и марки — в массиве `TASKS` в начале JS-блока. Менять контент = менять строку в `TASKS`, не трогая логику:

```js
const TASKS = [
  { id: 1, text: "Понюхать запах свежескошенной травы", stamp: "grass" },
  // ...
];
```

Финальные 20 фраз пишет автор курса. До этого — заглушки из master-ТЗ.

## Что НЕ делаем

- Загрузка фотографий участниц
- Бэкенд, аккаунты, соцслой
- Геймификация с давлением (таймеры, стрики)
- Адаптация под южное полушарие

## Визуал

- **Тон**: тихий, созерцательный, тактильный (Kinfolk / Cereal Magazine)
- **Палитра**: молочный (фон), лён (карточки), тёплый серый (текст/линии), приглушённый шалфейный (акцент/собрано)
- **Марки**: монолайн inline-SVG с перфорированным зубчатым краем
- **Анимация сбора**: fade текста + масштаб + лёгкий поворот + тень (клей)
- **Фон**: CSS-имитация бликов солнца на воде (чёрно-белая, низкий контраст)
- `prefers-reduced-motion`: упрощённые кроссфейды без вращений

## Доступность (quality floor)

- Ячейки — `<button>` с ARIA label («Собрать: <фраза>» / «Отлепить марку»)
- `aria-live` для счётчика и финала
- Видимый фокус-стиль для клавиатуры
- Контрастность читаема в рамках тихой палитры
