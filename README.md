# Летнее бинго

Минималистичный интерактивный сайт — дополнение к онлайн-курсу по фотографии.

В течение лета участница «собирает» приятные летние моменты: тапает на карточку с наблюдением, текст исчезает, на его месте появляется маленькая почтовая марка. Когда собраны все 20 — тихий финал: «Лето замечено».

**Главный принцип.** Это курс по фотографии, но практика намеренно без камеры — соединяться с моментами лета, не беря в руки телефон.

## Технологии

Один файл `index.html` — HTML + CSS + JS инлайн, без фреймворков и сборки. Прогресс хранится в `localStorage`. Деплой на GitHub Pages.
